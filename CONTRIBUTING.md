# Contributing to ACRD-GEMINI

Thank you for your interest in contributing to the Android Custom ROM Development (ACRD) Gemini toolset!

## How to Contribute

1.  **Report Bugs:** Use the GitHub Issue Tracker.
2.  **Suggest Features:** We welcome new ideas! Open an issue to discuss.
3.  **Submit Pull Requests:**
    *   Fork the repository.
    *   Create a new branch for your feature or fix.
    *   Ensure tests pass.
    *   Submit a PR with a clear description of changes.

## Development Setup

1.  Clone the repo.
2.  Install dependencies: `pip install -r requirements.txt`
3.  Run setup: `python3 setup.py`
4.  Run tests: `pytest tests/`

## Code Style

*   Follow PEP 8 for Python code.
*   Use `rich` for TUI elements.
*   Document new functions with docstrings.
